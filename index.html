Habilidades

HTML

HTML é uma linguagem de marcação, onde marcamos os elementos para definir quais informações a página vai exibir.

CSS

CSS, que significa Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de programação que permite definir a aparência e o layout de páginas web.

Figma

O Figma é uma ferramenta de design colaborativa e baseada na web, utilizada para criar interfaces de usuário (UI), experiências de usuário (UX), protótipos, e até mesmo apresentações interativas.

Javascript

JavaScript é uma linguagem de programação de script do lado do cliente, utilizada para adicionar interatividade e dinamismo a páginas web.

Tailwind

Tailwind CSS é um framework de estilo CSS que se concentra em fornecer um conjunto de classes de utilidade que podem ser aplicadas diretamente ao HTML para estilizar elementos.

React

React é uma biblioteca JavaScript de código aberto utilizada para construir interfaces de usuário (UI) em aplicativos web.

Bootestrap

Bootstrap é um framework front-end de código aberto utilizado para criar sites e aplicações web, facilitando o desenvolvimento de interfaces responsivas (adaptáveis a diferentes tamanhos de tela).

Typescript

TypeScript é uma linguagem de programação de código aberto, criada pela Microsoft, que se baseia no JavaScript e adiciona tipagem estática opcional.

Sass

SASS, que significa "Syntactically Awesome Style Sheets", é uma linguagem de extensão para CSS que oferece funcionalidades avançadas para a criação de estilos de páginas web.
